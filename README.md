# Day 18 – Social Sentiment & Trend Predictor (Project Presentation)

## 📌 Project Overview
The **Social Sentiment & Trend Predictor** is an Artificial Intelligence (AI) and Machine Learning (ML) based web application that analyzes social media text and classifies sentiments into **Positive, Negative, Neutral,** or **Irrelevant** categories. The project uses Natural Language Processing (NLP) techniques to provide fast and accurate sentiment analysis through a Flask-based web application.
